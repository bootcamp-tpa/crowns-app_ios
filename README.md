# crowns-app_ios
  
  
| ステップ |
| --- |
| 1. [CocoaPods](canvas.instructure.com/courses/1545672/modules)をインストールしてください |
| 2. `pod install` で依存ライブラリを追加しましょう。 |
| 3. goサーバーを起動しましょう |
| 4. アプリを起動して、起動からリーダーボード表示までのプログラムの動きを確認しましょう。 |
| 5. アプリ起動からリーダーボード表示までの動きの説明を、テキストファイルなどに図や文章などを追加して、PRを送ってください |
